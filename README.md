# jXFW

Java eXtendable FrameWork

## Website

[documentation](http://185.12.28.40/docsite/)

## Building jXFW

jXFW is built with [Apache Maven](http://maven.apache.org)


To build jXFW, run:

    mvn package

### Building environment
* Java 8 Development Kit (JDK)
* Maven 3.6.3
* Internet access
